Métodos de Arrays en JavaScript

map()

Crea un nuevo array aplicando una función a cada elemento del array original.

array.map(elemento => elemento * 2)

filter()

Crea un nuevo array con los elementos que cumplan una condición específica.

array.filter(elemento => elemento > 10)

reduce()

Reduce el array a un único valor acumulando los resultados de aplicar una función.

array.reduce((acum, elem) => acum + elem, 0)

find()

Encuentra y retorna el primer elemento que cumpla con una condición.

array.find(elemento => elemento.id === 5)

sort()

Ordena los elementos del array (modifica el array original).

array.sort((a, b) => a - b)

includes()

Verifica si un array contiene un elemento específico.

array.includes(5)

indexOf()

Retorna el índice de la primera ocurrencia de un elemento.

array.indexOf('valor')

some()

Verifica si al menos un elemento cumple con una condición.

array.some(elemento => elemento > 100)

every()

Verifica si todos los elementos cumplen con una condición.

array.every(elemento => elemento > 0)

slice()

Extrae una porción del array sin modificar el original.

array.slice(inicio, fin)

splice()

Añade, elimina o reemplaza elementos (modifica el array original).

array.splice(inicio, cantidad, ...nuevos)

indexOf()

Retorna el índice de la primera ocurrencia de un elemento.

array.indexOf('valor')

join()

Une todos los elementos del array en un string.

array.join(', ')

split()

Divide un string en un array usando un separador.

string.split(',')
